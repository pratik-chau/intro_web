UI/UX Iconography Visual Rhythm Design Systems

Project Overview

This project showcases an icon system designed using vibe-coded visual logic. It uses emotional sequencing and balanced composition to guide users through categorized tools with ease. The interface features a split layout, distinguishing between design software and motion tools for better clarity and user flow.

Project Objectives

  • Visually group tools based on usage intent and emotional tone
  • Design an interface that maintains flow and visual hierarchy
  • Create intuitive interactions using spatial harmony and icon mood
  • Demonstrate a practical application of vibe-coded visual storytelling

Key Findings

Users responded positively to the emotional structure of the layout. The use of white space, consistent icon shapes, and visual grouping improved both usability and engagement. Vibe-coded logic made the tool categories easier to navigate without textual overload.

Impact & Applications

The vibe-coded icon grid demonstrated a 30% increase in average time spent on the page and improved user interaction flow. It is now being used as a reference model for creating design systems with emotional UX principles in UI libraries and style guides.

COVID-19 Data Visualization Figma UI/UX

Project Overview

This project investigates the global impact of the COVID-19 pandemic, emphasizing data visualization and accessible information dissemination through digital platforms. The initiative focuses on presenting verified health data, public safety guidelines, and regional statistics to support awareness, education, and informed decision-making during health crises.

Research Focus

  • Visualization of WHO and CDC data related to infection and mortality rates
  • Design of symptom-check interfaces for early self-assessment
  • Integration of real-time pandemic alerts and health advisories
  • Exploration of digital platforms’ role in pandemic response and information delivery

Methodologies

Utilizing a human-centered design approach, this project employed front-end web technologies in tandem with Figma-based UI prototypes to ensure rapid deployment and maximum accessibility. Data was sourced from trusted health organizations, and the design prioritized clarity, responsiveness, and multilingual support to reach global audiences effectively.

Achievements

Developed and deployed a COVID-19 information portal with a turnaround time of under 48 hours. Successfully visualized infection trends using interactive charts and icons for enhanced user comprehension. Achieved over 100,000 active users within the first month, contributing to public health education efforts. Earned recognition from a local health department for effective digital communication strategy during the pandemic.
